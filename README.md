🪑 Furni – Modern Furniture Website

📌 Overview

Furni is a fully responsive furniture e-commerce front-end website built using HTML, CSS, and JavaScript. The project focuses on modern UI design, clean layout structure, and responsive user experience across devices.

This project demonstrates my ability to design and deploy a production-ready static website using Netlify.
